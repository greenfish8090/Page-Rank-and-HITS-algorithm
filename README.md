## How to use
### Create environment

```
conda env create -f environment.yml
```
### Run PageRank algorithm

```
python page_rank.py --file <filepath> --rtp (true|false)
```
### Run PageRank algorithm

```
python hits.py --graph <filepath> --query <term>
```
